# FMCG Supply Chain Analytics & Lead Time Forecasting

## 📌 Overview
This project focuses on analyzing FMCG supply chain data to identify delivery inefficiencies, optimize lead time, and improve inventory planning using data analytics, machine learning, and time-series forecasting.

---

## 🎯 Problem Statement
Supply chain operations often suffer from:
- High delivery delays
- Inefficient shipping modes
- Poor visibility into lead time variability
- Lack of predictive insights for planning

This leads to stock-outs, increased costs, and reduced operational efficiency.

---

## 🚀 Objectives
- Analyze supply chain data to identify inefficiencies
- Optimize lead time performance
- Predict delivery delays using machine learning
- Forecast future lead time trends
- Support data-driven inventory planning

---

## 🛠️ Tech Stack
- **Programming:** R  
- **Libraries:** Tidyverse, ggplot2  
- **Machine Learning:** Logistic Regression, Random Forest  
- **Time Series:** ARIMA  
- **Visualization:** Data Visualization Dashboards  

---

## 📊 Dataset
- 175,000+ FMCG supply chain records  
- Includes: orders, shipping modes, delivery status, lead time, and customer details  

---

## 🔍 Methodology

### 1. Data Preprocessing
- Data cleaning and handling missing values  
- Feature engineering for lead time and delivery status  

### 2. Exploratory Data Analysis (EDA)
- Identified delivery patterns and delays  
- Analyzed shipping mode performance  
- Visualized trends using ggplot2  

### 3. Machine Learning
- Built classification models to predict late deliveries  
- Models used:
  - Logistic Regression  
  - Random Forest (~85% accuracy)  

### 4. Time Series Forecasting
- Developed ARIMA model  
- Forecasted future lead time trends (~3.1 days stability)  

---

## 📈 Key Results
- Average lead time optimized to **3.09 days**  
- Identified **56.41% late deliveries**  
- Detected inefficiencies in shipping modes (up to **95% delay**)  
- Built predictive models with **~85% accuracy**  

---

## 💡 Insights
- Certain shipping modes significantly increase delay risk  
- Lead time variability directly impacts inventory planning  
- Predictive models help in proactive decision-making

- Enables **data-driven supply chain decisions**  
- Reduces delivery delays and operational inefficiencies
